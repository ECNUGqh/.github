# ECNUGqh Archive

本组织用于归档课题组毕业同学的论文、项目代码和相关资料。

## Naming Rule

- `paper-<year>-<abbr>[-author]`: paper source, manuscript, supplementary materials, and final PDF.
- `project-<year>-<abbr>[-author]`: project code, datasets notes, checkpoints notes, demos, and experiment scripts.
- Repository descriptions record the author, research direction, and archive scope.
- Repository topics provide searchable labels such as `archive`, `paper`, `project`, `author-*`, `alumni-*`, and `direction-*`.

## Maintenance

When adding a new archive repository:

1. Use the naming rule above.
2. Add a clear repository description.
3. Add searchable topics.
4. Add one row to the corresponding table in this README.
