# ECNUGqh Archive

本组织用于归档课题组毕业同学的论文、论文代码和相关资料。

## Naming Rule

- `paper-<year>-<abbr>`: paper source, manuscript, supplementary materials, final PDF, and paper-related code.
- Repository descriptions record the author, research direction, and archive scope.
- Repository topics provide searchable labels such as `archive`, `paper`, `author-*`, `alumni-*`, and `direction-*`.

## Maintenance

When adding a new archive repository:

1. Use the naming rule above.
2. Add a clear repository description.
3. Add searchable topics.
4. Add one row to the corresponding table in this README.
