# ECNUGqh 归档

本组织用于归档课题组毕业同学的论文、论文代码和相关资料。

## 命名规则

- 仓库名直接使用论文缩写，例如 `DPSN`、`PGDTS`。
- 仓库描述记录作者、研究方向和归档内容。
- 仓库标签用于检索，例如 `archive`、`paper`、`author-*`、`alumni-*` 和 `direction-*`。

## 维护方式

添加新的归档仓库时：

1. 使用上面的命名规则。
2. 补充清晰的仓库描述。
3. 添加便于检索的仓库标签。
4. 在本页表格中增加对应条目。
